### Deploy in GitHub
```bash
ng add angular-cli-ghpages
ng deploy
```
or
```bash
ng add angular-cli-ghpages
ng build --prod --output-path docs --base-href /MITRO/
ng deploy --no-build
```

```bash
git reset --hard HEAD~1
```
